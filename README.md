# TODO List

一个简洁的待办事项应用，帮助你管理日常任务。

## 功能

- 添加/删除待办事项
- 标记任务完成状态
- 简洁美观的界面
- 响应式设计，支持移动端

## 快速开始

```bash
npm install
npm start
```

访问 http://localhost:3000 即可使用。

## Docker 部署

```bash
docker pull ghcr.io/your-username/todolist:latest
docker run -d -p 3000:3000 ghcr.io/your-username/todolist:latest
```

## 技术栈

- Node.js
- Express
- HTML5 / CSS3
