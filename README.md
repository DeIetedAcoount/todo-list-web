# TODO List

A simple todo application to help you manage daily tasks.

## Features

- Add/delete todo items
- Mark tasks as complete
- Clean and beautiful interface
- Responsive design, mobile friendly

## Quick Start

```bash
npm install
npm start
```

Visit http://localhost:3000 to use.

## Docker Deployment

```bash
docker pull ghcr.io/your-username/todolist:latest
docker run -d -p 3000:3000 ghcr.io/your-username/todolist:latest
```

## Tech Stack

- Node.js
- Express
- HTML5 / CSS3
